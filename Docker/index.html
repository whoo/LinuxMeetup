Mon premier container

  • Simple

    docker run -it alpine

    docker run -it debian

  • Plus complexe [ avec interactions]

    docker run -it -p8080:80 owncloud

    docker run -it -v /tmp:/tmp debian ls

  • Encore plus

    docker run -it -v etc:/root/elt –name dd debian

    docker run -it -v etc:/root/elt --link dd:other debian

    Sur les systèmes autres que linux, les echanges sont plus complexes à cause de la couche de virtualisation
