  • iptables permet de proteger les ports depuis le réseau. Source | Destination | sport | dport | limit

Un firewall ... il y a deux sens.

Exemple iptables

# iptables -A INPUT -m state --state ESTABLISHED,RELATED  -j ACCEPT
# iptables -A OUTPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
# iptables -A OUTPUT -p tcp --dports 22,80,443            -j ACCEPT
# iptables -A OUTPUT -p udp --dports 53                   -j ACCEPT

# iptables -A INPUT   -i lo                               -j ACCEPT
# iptables -A OUTPUT  -o lo                               -j ACCEPT

# iptables -P INPUT DROP
# iptables -P OUTPUT DROP

LUKS

Linux Unified Key Setup permet de chiffrer les systèmes de fichiers sous linux.
En fonction du système proposé, le système demandera une passphrase, ou un fichier de clef.

cryptsetup luksFormat -c aes-xts-plain64 -s 512 -h sha512 /dev/sss

Demo

Plusieurs clefs/mot de passe, fichier sur une clef usb, ... Il suffit de suppimer les premiers octets du stockage pour rendre inutilisable.
